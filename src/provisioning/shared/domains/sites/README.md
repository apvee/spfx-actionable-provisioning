# Sites Domain Helpers

Internal utilities for SharePoint site operations.

## Purpose

Provides reusable functions for site-related operations.

## Files

| File | Description |
|------|-------------|
| index.ts | Barrel export |

## Planned Additions

- Site lookup utilities
- Site permission probing

## Adding New Helpers

1. Create a new `.ts` file in this folder
2. Add `@internal` JSDoc tag
3. Export from `index.ts`
